Example trace prop:
```js static

 traces={
            [
                {
                    pv:'pva://testIOC:BO1',
                    yAxis:0
                },
                {   pv:'pva://testIOC:amplitude',
                    yAxis:1
                },
                {   pv:'pva://testIOC:BO2',
                    yAxis:2
                }
            ]
        }

```
Note: for plotly to display the traces correctly the yAxis:0 always needs to be assigned.

Example yAxes prop:
```js static

 traces={
           yAxes={
            [
                {
                    title: "BO1",
                },
                {
                    title: "Amplitude",
                },
                {
                    title: "BO2",
                },

            ]}
        }

```
