

TextUpdate example connection to a SoftChannel EPICS AI pv with custom label, precision and units:

```js
<TextUpdate  
   pv='pva://$(device):test$(id)'
   macros={{'$(device)':'testIOC','$(id)':'1'}}
   label={"My value is "}
   units={"mA"}
   usePrecision={true}
   prec={2}
/>
```
